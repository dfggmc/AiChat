# AiChat | 在线AI聊天 
 
使用MDUI前端框架构建，提供与AI的即时聊天功能。
 
## 功能概述
 
- [ ] 独立设置聊天
- [x] 聊天记录
- [x] 聊天列表
 
> **注意:** 默认数据通过调用各网站的API接口获取，某些接口可能不够稳定，存在随时可能失效的风险。
